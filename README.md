## ✨ Version 1 Revamp (Premium Dark Glass Edition)
The entire user interface has been overhauled to a premium, futuristic dark mode aesthetic.
*   **Pure Black Canvas:** Deep, immersive pure black (`#000000`) background across all routes.
*   **Glowing Glass Tiles:** All dashboard modules, attendance cards, exams, and hostel sections now feature a sleek dark purple translucent gradient (`from-[#1c0f30]/80 to-black`) with backdrop blur.
*   **Premium Interactions:** Smooth micro-animations, subtle scaling effects, and reactive neon purple borders on hover.
*   **Consistent Design Language:** From the Proctor profile card down to the detailed exam schedule popups, the entire application speaks the same high-end, modern design language.
